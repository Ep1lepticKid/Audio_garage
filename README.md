Аудио Гараж — интернет-магазин звукового оборудования
Аудио Гараж — это полнофункциональный интернет-магазин для продажи студийного и сценического звукового оборудования. Проект выполнен в уникальном ретро-неоновом стиле и включает как публичную часть для покупателей, так и административную панель для управления контентом.


✨ Особенности проекта
Для покупателей
Главная страница с параллакс-эффектом и изображением гаража

Каталог категорий с плиточным отображением и закругленными карточками

Страница категории с фильтрацией по цене и наличию

Карточка товара с галереей изображений, характеристиками и описанием

Корзина с проверкой доступного остатка на складе

Оформление заказов с формой контактных данных

Раздел статей с полезными материалами

Страница контактов с интерактивной картой

Для администраторов
Дашборд с реальной статистикой (товары, категории, заказы, статьи)

Управление товарами (CRUD, загрузка изображений)

Управление категориями (сортировка, генерация slug)

Управление заказами (просмотр, изменение статуса, история)

Управление статьями (создание, публикация)

Загрузка изображений с эффектом сепии

Установка и запуск
1. Клонирование репозитория
bash
git clone https://github.com/твой-логин/audio-garage.git
cd audio-garage
2. Настройка бэкенда
bash
cd backend
npm install
cp .env.example .env  # настрой переменные окружения
npm run dev
3. Настройка фронтенда
bash
cd frontend
npm install
cp .env.example .env  # настрой переменные окружения
npm start
4. Создание базы данных
sql
CREATE DATABASE audiogarage_db;
-- выполни SQL-скрипты из папки database/
🐳 Запуск через Docker
bash
# Сборка и запуск всех контейнеров
docker-compose up -d --build

# Остановка
docker-compose down

🎨 Дизайн-концепция
Проект выполнен в уникальном ретро-неоновом стиле:

Цветовая палитра:

Тёмно-серый фон (#2b2a24) — атмосфера гаража

Приглушённый зелёный (#7a914b) — отсылка к старому металлу

Неоновый зелёный (#00ff88) — акценты, свечение

Логотип:

Буква «А» в слове «ГАРАЖ» перевёрнута и покачивается, будто держится на одном гвозде

Буквы «Г» и «РАЖ» мерцают неравномерно, имитируя старую неоновую вывеску

Типографика: моноширинный шрифт Courier Prime для заголовков

🔐 Аутентификация
JWT-токены для защиты маршрутов админки

bcryptjs для хэширования паролей

Роли пользователей: admin, manager

📦 Основные функции API
Метод	Endpoint	Описание

GET	/api/products	Список товаров с пагинацией и фильтрами

GET	/api/products/:id	Детальная информация о товаре

POST	/api/products	Создание товара (admin)

PUT	/api/products/:id	Обновление товара (admin)

DELETE	/api/products/:id	Удаление товара (admin)

GET	/api/categories	Список категорий

GET	/api/orders	Список заказов (admin)

POST	/api/orders	Создание заказа

POST	/api/auth/login	Вход в систему
